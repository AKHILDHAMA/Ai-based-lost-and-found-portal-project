# AI-Based Lost and Found Portal

An intelligent web application that helps users report, search, and recover lost items using AI-powered matching, chatbot support, and map integration.

---

## Features

- User Registration & Login
- Report Lost Items
- Report Found Items
- AI-Based Item Matching
- AI Chatbot Assistance
- Interactive Map Integration
- Recovery Stories Section
- Live Activity Feed
- Image Upload Support
- Responsive User Interface

---

- Git & GitHub

---

## Project Workflow

1. User logs into the portal
2. User reports a lost or found item
3. System stores item details in database
4. AI compares item descriptions, images, and locations
5. Matching results are shown to users
6. Users connect and recover items
